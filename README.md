## Lessons Learned

As I worked through the initial Comp I could tell the intermediate css CYOA would give
me a hard time. As I worked to get the comp to match the visual spec I used pixels to size many of the
elements.  Although I went back and worked to change most of them to percentages, the initial coding still caused a lot of
trouble for resizing the display settings.
